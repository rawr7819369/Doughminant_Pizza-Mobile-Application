# Developer Images

Place developer profile images in this directory.

## Image Requirements:
- **Format:** JPG, PNG, or WebP
- **Size:** Recommended 400x400px (square)
- **File Size:** Keep under 200KB for optimal performance
- **Naming:** Use the exact filenames specified in `about-us.page.ts`

## Required Files:
1. `alex-johnson.jpg` - Lead Developer
2. `sarah-martinez.jpg` - Frontend Developer
3. `michael-chen.jpg` - Backend Developer
4. `emily-rodriguez.jpg` - UI/UX Designer
5. `david-kim.jpg` - Quality Assurance

## Notes:
- If an image is not found, the app will automatically fall back to the icon
- Images should be professional headshots or profile photos
- Ensure good lighting and clear backgrounds for best results
